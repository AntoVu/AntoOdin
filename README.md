# AntoOdin
Simple odin addon for more features I wanted
## Features
Wardrobe Hotkeys (A)
 - Auto close after use
 - Play sound on equip

Queue Wardrobe
 - Auto equip wardrobe on keybind

Kuudra Auto GFS
 - Get TAP/TWAP after build

CPS Display
 - Displays cps for left, right, or both

Pearl Refill
 - Auto refill pearls after set amount
